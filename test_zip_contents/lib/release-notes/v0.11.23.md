# Release Notes - v0.11.23

**Release Date:** September 22, 2025  
**Version:** 0.11.23  
**Previous Version:** 0.11.22

## New Features

- Enable Git AI analytics for all users with external filepath hashing; improve periodic collection behavior (Analytics)
- Add `/memory reflect` command to analyze the current session and surface insights
- Support referencing instruction markdown files directly in prompts via `!filename.md`
- Add Serve API endpoint and CLI argument to set/update the site URL
- Add session ID header to API requests for better traceability
- Log average evaluation metrics to MLflow in Code Conseil evaluators

## Bug Fixes

- Restore image/document viewing by only truncating string tool results (Nemo)
- Correct healthcheck and MCP startup states; include 'unknown' state and return 503 on failure (Serve)
- Fix auto-restore behavior in Serve mode
- Register Bitbucket MCP server only when scoped token enables BBC tools
- Show real user initial prompt in session details; ignore synthetic workspace message (UI)

## Engineering Improvements

- Add Quickstart guide and automate documentation updates via Bitbucket Pipelines
- Expand diagnostics evaluation dataset and add additional test coverage; filter evaluator to error/warning severities (Code Conseil)
- Upgrade rich to 14.1.0; remove usage of private Console._live API in CLI callback/tests
- Improve tool payload compression
- Gate Serve v2 endpoints to internal users; expose v3 for external CLI and add route filtering tests
- Log metrics to MLflow for evaluation runs
