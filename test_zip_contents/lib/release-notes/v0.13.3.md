# Rovo Dev CLI v0.13.3 (2025-11-22)

## Summary
Major feature release introducing voice input support, enhanced subagent capabilities, Fireworks AI provider, and numerous stability improvements across Windows and shell interactions.

## New Features
- **Voice Input Support**: Added F9 keybinding for voice-to-text input using faster-whisper
- **Enhanced Editor Integration**: Added F2 keybinding for external editor prompt crafting
- **Fireworks AI Provider**: Added support for Fireworks AI models
- **Subagent Memory**: Subagents can now load and utilize memory files
- **Usage Command in Serve Mode**: Added /usage command support for serve mode
- **Haiku 4.5 Migration**: Implemented Claude Haiku 4.5 migration for internal users

## Bug Fixes
- Fixed Windows SSL compatibility by setting additional CA bundle environment variables
- Resolved bash/PowerShell processes hanging on stdin input
- Improved ask mode compatibility and error handling for different AI models
- Fixed session feedback collection in non-interactive and YOLO modes
- Handled keyboard interrupts buried inside exception groups
- Fixed agent mode transitions and state management

## Engineering Improvements
- Migrated multiple slash commands (jira, usage, sessions, models) to async architecture
- Enhanced documentation with missing V3 API endpoints and permission resolution
- Improved CI/CD with documentation automation and agent syntax migration
- Added evaluation automation utilities and testing best practices
- Enhanced workspace view handling for acra_mini agent
- Improved Git AI v2 binary resolution and bundle layout
