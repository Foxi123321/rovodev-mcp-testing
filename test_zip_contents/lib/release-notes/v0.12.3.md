# Release Notes - v0.12.3

**Release Date:** October 10, 2025  
**Version:** 0.12.3  
**Previous Version:** 0.12.2

## New Features

- Changelog: add `changelog` command for generating change summaries
- Commit workflow: add commit prompt helper for guided commit messages
- Analytics: include actorId analytics data in events
- Analytics: include beta user status in analytics payloads
- CLI: extended help now available for sub-commands

## Bug Fixes

- Usage: raise exceptions properly from `usage` command to surface failures
- Site selection: remove redundant logic and ensure cloudId is included in usage cache key
- Health: prevent incorrect transient healthy status
- Proxy/Sandbox: workaround for v3 sandbox proxy requests
- Jira: fix scoped token handling
- Confluence: prevent agent from using HTML for Confluence content
- Windows: fix test failures and pin correct Python version

## Engineering Improvements

- Analytics: use a fixed-size thread pool for analytics event processing
- LSP/Analytics: add LSP analytics tracking and unknown command handling
- Specs: bump OpenAPI spec version to v3.0.3
- CLI: small fixes and code cleanups; remove duplicate tests
- Memory: support custom file path for memory reflect in Rovo Dev
- Prompt handling: fix command-line argument handling for prompt file syntax
