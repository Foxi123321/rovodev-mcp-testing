# Release Notes - v0.11.17

**Release Date:** January 9, 2025  
**Version:** 0.11.17  
**Previous Version:** 0.11.16

## New Features

- **Subagents Support**: Added comprehensive subagents functionality with MCP server integration and model selection capabilities, enabling more sophisticated multi-agent workflows
- **GPT-5 Integration**: Set GPT-5 as the default model with new auto model selection option for improved AI capabilities
- **OpenAI Responses API**: Implemented support for OpenAI responses API with dedicated model handling for enhanced AI interactions
- **Agent Lifecycle Events**: Added comprehensive agent lifecycle events system for better monitoring and control of agent operations
- **Status Command**: Introduced new status command functionality to provide better visibility into system state
- **AGENTS.md Support**: Added support for AGENTS.md files to improve project-specific agent configuration and documentation

## Security Fixes

- **CRITICAL Path Expansion Vulnerability**: Fixed critical security vulnerability that could lead to home directory deletion through improper path expansion handling
- **Command Allowlist Bypass**: Resolved security issue with default command allowlist bypass via buried bare git repositories

## Bug Fixes

- **LSP Improvements**: Fixed LSP server to report warnings instead of errors for better development experience
- **Windows Compatibility**: Resolved multiple Windows-specific issues including tilde expansion tests, Unicode memory errors, and PowerShell tool functionality
- **Bash Command Safety**: Enhanced bash command execution with improved safety checks and prevented dangerous rm command execution
- **Error Handling**: Improved error handling across multiple components including MCP exception logging, stacktrace sanitization, and help command functionality
- **Runtime Stability**: Fixed AnyIO cancellation scope runtime errors and suppressed unnecessary MCP SDK warning logs

## Engineering Improvements

- **Code Cleanup**: Removed legacy ACRA-mini prompt template and streamlined codebase
- **Jira Enhancement**: Enabled creating Jira tickets with custom fields for better project management integration
- **Analytics Enhancements**: Added AI result viewed event tracking, DevAI Axis span processor, and analytics copy shortcut functionality
- **Testing Infrastructure**: Added fresh install tests and improved test execution performance
- **macOS Sandbox**: Enhanced macOS sandbox functionality for better security and compatibility
- **User Experience**: Updated startup banner messaging and improved internal command error messages
