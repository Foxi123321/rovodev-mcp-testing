# Release Notes - v0.12.1

**Release Date:** October 6, 2025    
**Version:** 0.12.1  
**Previous Version:** 0.12.0

## New Features

- Usage: display "unlimited" when monthly total is unlimited
- Entitlements: centralize user state and inject AI Gateway headers per request
- Providers: use low reasoning effort by default for improved cost/perf balance

## Bug Fixes

- Serve: handle None tool_name in MCP tool expansion for OpenAI streams
- Exceptions: correct CTA link formatting and centralize processing
- Exceptions: handle missing CTA link placeholder with CTA present in template
- Windows: fix get file list path handling in sandboxed environments
- Windows CI: prevent uv from writing to cache to stabilize pipelines
- Windows: rename bash config to powershell on Windows environments
- Sandbox: improve warning messaging for Windows sandbox restrictions
- Subagents: fix tool filters to account for MCP tool compression

## Engineering Improvements

- Workspace view: improve rendering and usability
- Tool call events: include permission information and minor event schema tweaks
- GUI: minor build improvements
- Logging: configure logging initialization for consistency across modules
- Misc: merge readiness branches and minor cleanups
