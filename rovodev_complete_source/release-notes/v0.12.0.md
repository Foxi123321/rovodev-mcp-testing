# Release Notes - v0.12.0

**Release Date:** October 3, 2025  
**Version:** 0.12.0  
**Previous Version:** 0.11.29

## New Features

- Analytics: log built-in agent names in agent run events (exclude user-defined)
- Entitlements: route existing beta users to legacy endpoints and propagate beta status via environment variable and header

## Bug Fixes

- Usage: handle None/null values in balance counts from the check endpoint
- Analytics: compute actorIdHash from user ARI instead of raw user_id
- Site selection: correctly detect beta users by checking isBeta at the site level

## Engineering Improvements

- Documentation: add permissions documentation and CI scaffolding for automated permission doc PRs
- Documentation: add v0.11.29 release notes
