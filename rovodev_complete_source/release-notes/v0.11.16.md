# Release Notes - v0.11.16

**Release Date:** September 1, 2025  
**Version:** 0.11.16  
**Previous Version:** 0.11.15

## New Features

- **Right-to-Left Integrations**: Added proactive integration suggestions for Jira, Confluence, and Bitbucket. The agent now suggests creating work items, documentation, or pull requests after completing significant work
- **MCP Server Management**: Added ability to manually enable or disable MCP servers with confirmation dialogs and proper error handling
- **Enhanced Jira Integration**: Enabled getting and updating any fields from Jira issues, providing more comprehensive issue management capabilities
- **Bash Command Permission Management**: Implemented granular permission management for bash subcommands with improved parsing and validation
- **CLI Startup Enhancements**: Improved startup design and user experience with better messaging and flow
- **IDE Analytics**: Added analytics tracking for IDE command selection to better understand user interactions

## Bug Fixes

- **Serve Mode Shutdown**: Fixed issues with serve mode shutdown process and added dedicated `/shutdown` endpoint for graceful server termination
- **Bash Command Parsing**: Improved bash command splitting to properly handle quoted arguments and complex command structures
- **Tool Call Compatibility**: Updated to use `direct_call_tool` to maintain compatibility with Pydantic AI 0.6.2
- **Planning Tool**: Disabled planning tool for initial serve chat to improve user experience
- **Git Error Handling**: Added proper error handling for git operations in code review evaluation

## Engineering Improvements

- **Test Quality**: Replaced ineffective tests with comprehensive system prompt verification tests that actually validate functionality
- **Git Metadata**: Added capability to save git metadata locally and to MLFlow for better tracking and analysis
- **Windows Compatibility**: Skipped problematic git tests on Windows to improve CI reliability
- **API Documentation**: Updated serve API documentation with missing endpoints and comprehensive examples
