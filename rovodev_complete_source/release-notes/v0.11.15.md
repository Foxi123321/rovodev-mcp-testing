# Release Notes v0.11.15

**Release Date:** January 27, 2025  
**Version:** 0.11.15

## New Features

- **Atlassian Experimental MCP Server**: Added atlassian-exp MCP server to binary distribution with dynamic configuration support for enhanced Atlassian integrations.
- **IDE Extension Auto-Install Analytics**: Implemented comprehensive analytics tracking for automatic IDE extension installation with success/failure monitoring.
- **Reproduction and Diff Rater Agents**: Introduced new agents for SWE-bench evaluations including reproduction agent for test case generation and diff rater agent for solution comparison.
- **Enhanced Jira Integration**: Redesigned Jira integration with improved user experience, better configuration flow, and disable functionality.
- **SWE-Bench Evaluation Support**: Added SWE-Bench eval server setup script for automated evaluation workflows.

## Bug Fixes

- **Git-AI Analytics Crash**: Fixed AttributeError crashes in git-ai analytics by properly handling JSON-serialized stats from Logfire.
- **CLI Startup Warning**: Prevented syntax warnings on rovodev CLI startup by fixing string formatting in agent prompts.
- **Stale LSP Diagnostics**: Implemented request ID tracking to prevent stale LSP diagnostics from being published.

## Engineering Improvements

- **Message History Persistence**: Enhanced auto-pruning to persist changes to message history for better conversation state management.
- **LSP Agent Improvements**: Refined LSP agent prompt with better diagnostics guidelines and improved error handling.
- **Evaluation Filtering**: Improved similarity score calculation by filtering test/docs files before evaluation.
- **Code Quality**: Applied consistent string formatting and removed syntax warnings across the codebase.
