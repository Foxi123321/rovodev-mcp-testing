# Release Notes - v0.11.11

**Release Date:** August 22, 2025  
**Version:** 0.11.11

## Summary

This release enhances analytics capabilities with response data collection, introduces Experience ID (xid) support across the CLI, and includes automated git tagging for releases. It also fixes system prompt handling and improves git context structure for better analytics.

## New Features

- **Response Data Collection**: Added response data collection to agent_run analytics when prompt collection is enabled, including response content and length metadata
- **Experience ID (xid) Support**: Introduced `--xid` flag for tagging requests and analytics across CLI commands, with default value `rovodev-cli` and deprecated `--application-id` as alias
- **Automated Git Tagging**: Added automated git tagging for releases that triggers on version.py changes to create version tags
- **Code Review Evaluator**: Added comprehensive Code Review code-conseil evaluator with similarity scoring and ML Flow integration

## Bug Fixes

- **System Prompt Handling**: Fixed issue where not all system prompts were being used properly in agent runs (RDA-337)
- **Git Context Structure**: Improved git context structure and naming for better analytics, moving git context to top-level attributes with granular fields

## Engineering Improvements

- **Analytics Enhancement**: Enhanced git context with workspace, repository_name, branch fields and privacy protection for external users
- **Header Integration**: Added Experience ID headers for DevAI Axis and AI Gateway requests with `X-RovoDev-Xid` header
- **Code Organization**: Refactored Bitbucket CODEOWNERS file and renamed teams config file for better maintainability
- **Memory Management**: Improved memory prompt handling with proper context refresh at agent run start
