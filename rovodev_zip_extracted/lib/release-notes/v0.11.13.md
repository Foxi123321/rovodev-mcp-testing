# Release Notes v0.11.13

**Release Date:** January 14, 2025  
**Version:** 0.11.13

## New Features

- **LOC Metrics for Git-AI Stats**: Added comprehensive Lines of Code (LOC) metrics to git-ai statistics for enhanced PR impact analysis. The system now tracks AI-generated, human-written, and unknown authorship lines separately, enabling better analytics on code contribution patterns.

## Bug Fixes

- **Config File Stability**: Fixed config file reversion issues during test execution that could cause configuration inconsistencies.

## Engineering Improvements

- **Circular Import Resolution**: Removed circular import dependencies in the codebase by restructuring JiraProject model location and improving module organization.
- **Enhanced Analytics**: Integrated LOC calculation into the main git-ai stats collection pipeline with comprehensive deduplication and categorization logic.
