# Release Notes - v0.11.25

**Release Date:** September 29, 2025  
**Version:** 0.11.25  
**Previous Version:** 0.11.24

## New Features

- Make serve mode publicly available to all users (CLI)
- Add staging/server environment parameter for site entitlement testing (CLI)
- Auto-run MidConversationPruner when cache expires to reduce context size (CLI/Nemo)
- Allow recursive instruction chaining (CLI UI)
- Implement consistent green color scheme across CLI UI elements (CLI UI)

## Bug Fixes

- Fix instruction injection bug and add regression test
- Prevent 'Agent interrupted' message panel from being mangled; ensure newline flush
- Prevent use of PowerShell tool when sandboxing is enabled; add Windows startup warning
- Move early debug logs to after logging is configured to avoid console noise
- Update sites endpoint URL and beta user check to latest backend contract
- Improve Windows compatibility for environment propagation test in run command
- Handle NaN values during code review evaluation to avoid metric errors

## Engineering Improvements

- Future-proof ACLI/rovodev binary download process in CI
- Analytics: add agent run CBP usage-tracking mapper and related metrics
- Site entitlement: prefer site URL in config and set CLOUD_ID via environment
- Maintain V2 endpoints as internal, V3 as public; adjust tests to reflect API strategy and fix Windows pipeline
- Help system: inject CLI help content as a user message instead of system prompt for AI-powered help
- Documentation/CI: centralize PR creation instructions; simplify doc-generation guides; automate MCP docs
- UX: add consistent cancel hint across spinners and improve startup spacing
