# Release Notes - v0.11.20

**Release Date:** September 16, 2025  
**Version:** 0.11.20  
**Previous Version:** 0.11.19

## New Features

- Collect Git AI stats on shutdown for internal users
- Add HJSON configuration support
- Add restore option to Serve mode
- Add built-in BBC MCP server support
- Add dynamic config help content
- Support diagnostics dataset, evaluator, and runner in Code Conseil
- Support configured runner types in Code Conseil
- Support site entitlement and new user workflow in CLI client (RDA-385)
- Add analytics events for DevAI Axis
- Compress tool payloads for improved performance

## Bug Fixes

- Add safe shutdown signal handlers to avoid async SystemExit errors (Git AI stats)
- Prevent session fork metadata corruption
- Improve file deletion safety to avoid accidental removals
- Prevent concurrent subagent execution
- Remove double slashes when resolving workspace paths
- Fix prefix numbering in help/prompt output
- Modify workspace prompt to prevent weird output
- Ensure valid MCP tool prefixes are enforced

## Engineering Improvements

- Redirect new users to dashboard; hide daily usage behind entitlement flag (usage)
- Small refactor in comment generation code
- Move LSP models next to the agent for better cohesion
- Add release notes subagents to CI
- Add Conseil documentation
- Merge previous release notes pull request
