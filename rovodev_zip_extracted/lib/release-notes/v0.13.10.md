# Release Notes - v0.13.10

**Release Date:** December 2, 2025  
**Version:** 0.13.10  

## New Features

- **Christmas-themed Banners**: Added festive Christmas-themed ASCII art banners with dynamic theme configuration support
- **Deep Planning CLI Flag**: Added `--enable-deep-plan` flag to serve command for overriding config file settings
- **Config File Path Display**: Enhanced config command to display the current config file path for better transparency
- **Tilde Expansion for Prompts**: Added support for tilde (~) expansion when loading prompt file paths
- **Git AI v2 Checkpoints**: Introduced v2 checkpoints with agent-v1 integration and enhanced analytics tracking

## Bug Fixes

- **Security Fix**: Prevented auth tokens from being passed as command line arguments to enhance security
- **GLIBC Conflicts**: Fixed MCP server conflicts by excluding library path environment variables
- **Subagent Failures**: Preserved toolset references to prevent subagent failures when MCP servers encounter errors
- **MCP Error Messages**: Prevented misleading success messages when MCP server startup fails
- **Interrupt Handling**: Fixed handling of interruptions when processing deferred tool requests
- **GUI Shutdown**: Improved Ctrl+C signal handling for graceful GUI shutdown

## Improvements

- **MCP Spinner Updates**: Enhanced MCP server initialization with earlier spinner updates for better user feedback
- **Evaluation Robustness**: Improved evaluation system with Docker checks and better error handling
- **GUI Logging**: Refined logging behavior by disabling stderr logging in GUI mode
- **ACP Documentation**: Added comprehensive ACP (Agent Control Panel) documentation with CI automation
