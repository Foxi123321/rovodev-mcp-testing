# Release Notes - v0.11.9

**Release Date:** August 21, 2025  
**Version:** 0.11.9

## Summary

This release introduces the ability to disable Jira integration, automatic IDE extension installation, and various improvements to the developer experience.

## New Features

- **Jira Integration Toggle**: Added support for disabling Jira integration in the config file, giving users more control over their workflow
- **Automatic IDE Extension Installation**: Implemented auto-installation for Rovo Dev IDE extensions in VS Code, Cursor, and Windsurf

## Bug Fixes

- **IDE Detection**: Updated IDE check logic for auto-install IDE plugin to better handle different IDE variants and paths
- **MCP CLI Command**: Restored missing MCP CLI command functionality

## Engineering Improvements

- **Jira Prompts**: Cleaned up and simplified Jira-related prompts for better user experience
- **Comment Ranker**: Improved prompts and cleanup for comment ranker tool
- **CI/CD Pipeline**: Moved auto PR steps to main branch pipeline for better workflow
- **Code Ownership**: Added CODEOWNERS and team definitions for better code review process
