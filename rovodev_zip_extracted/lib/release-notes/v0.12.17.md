# Rovo Dev CLI v0.12.17 (2025-11-10)

## Summary
- Maintenance release with CI improvements to release notes automation, multiple bug fixes across CLI and agents, and small feature additions including XID parameter support in run command and a new MCP Scout capability to fetch recent PRs.

## New Features
- feat(mcp-scout): add tool to retrieve recent PRs for context enrichment (8382e32)
- feat(run): add/experiment with XID parameter support in run command (38031e5)
- feat(ui/terminal): highlight commands in terminal output for better readability (bdb852b)

## Bug Fixes
- fix(run/sessions): correct session workspace filtering behavior (caab1dd)
- fix(perf): address high CPU usage in background processing (91f6ffe)
- fix(config/network): standardize host resolution to localhost where applicable (85636cf)
- fix(ui/startup): correct warning panel formatting during startup spinner (1958400)
- fix(env): make CLI usable when mesh-dependent AI gateway env var is set (e82680b)
- fix(telemetry): disable feedback survey YOLO flag (b5de67c)
- fix(telemetry): adjust feedback survey toggles (9e25d70)

## Engineering Improvements
