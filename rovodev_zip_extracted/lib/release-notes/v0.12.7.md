# Release Notes - v0.12.7

**Release Date:** October 20, 2025  
**Version:** 0.12.7  
**Previous Version:** 0.12.6

## New Features

- Analytics: robust user prompt collection via callback; mapper updates; enabled in run and serve
- Analytics: add session feedback callback, config support, and mapping with tests
- UI: add Ctrl-F/Right to accept suggestions; Tab inserts 4 spaces
- Analytics: include authMethod in usage tracking events
- CLI: add Vim-style :q exit command
- Smart Tasks: add foundation with filesystem discoverer, /tasks command, baseline prompts
- Analytics: track LLM request count and behaviourId; mark errors on interrupt

## Bug Fixes

- Models: correct Bedrock Claude Sonnet 4.5 context limits; add Statsig-based fallback
- Fallback Model: broaden EOF JSON parse error handling to prevent crashes
- API Gateway: add retries and 429 rate limit handling with tenacity
- Comment Generation: ensure artifacts saved on error; refactor creation flow

## Engineering Improvements

- CI/Artifacts: remove OpenAPI spec and Shipyard artifact uploads
- Docs: add v0.12.6 release notes
