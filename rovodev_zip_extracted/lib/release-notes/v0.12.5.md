# Release Notes - v0.12.5

**Release Date:** October 15, 2025  
**Version:** 0.12.5  
**Previous Version:** 0.12.4

## New Features

- Subagents: add system subagents with ancestor directory traversal; support internal-only built-ins
- CLI/UI: customizable command prompt
- Analytics: add behaviour-id header propagation

## Bug Fixes

- CLI: return meaningful exit codes on crash; add exit code mapping and tests
- UI: prevent prompt moderation from blocking input flow
- UI: unify footer style for consistent appearance
- Subagents: fix duplicate path warning
- Billing/Usage: handle missing billing cloud ID gracefully
- IDE Integration: fix auto-installation errors in IDE integration
- Windows: fix stdout/stderr pipe handling and patch Windows-specific issues
- Entitlements: append site URL to entitlement error messages without ETA link
- Auth: perform entitlement check when using API token

## Engineering Improvements

- UI Architecture: implement component class system; migrate session menu and user input panel
- Caching: improve stale cache handling
- Tests: relax cache timing assertion to deflake tests
- Tooling/CI: add PR creation script scaffolding; reverse changelog order for readability
- Docs/Specs: tombstone unused OpenAPI Shipyard spec
- Observability: log user and cloud ID for debugging
