# v0.12.16 (2025-11-06)

A minor feature and fixes release for the Rovo Dev CLI and related packages.

## Highlights
- New: Add `/config` command with auto-restart when configuration changes
- New: Ask mode available in Serve API
- UX: Improved input navigation with vi/emacs-style keys; reduced keybinding latency
