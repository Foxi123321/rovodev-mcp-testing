# Rovo Dev CLI v0.12.15 (2025-11-05)

## Summary
- New features including Google AI Gateway provider support, enabling the /ask subcommand, and partial suggestion acceptance in the chat input. Multiple bug fixes across CLI, MCP, Nemo, and Nautilus. Documentation updates included.

## New Features
- feat(nemo/google): add AI Gateway Google provider and switch google models to GoogleModel with improved error handling (f339f21)
- feat(run): enable /ask subcommand with read-only agent mode and MCP tool restriction updates (2a7af47)
- feat(ui/chat-input): add word-by-word partial suggestion acceptance via Esc-F/Right and supporting text utils/tests (ad014c7)
- feat(analytics): include Rovo Dev entitlement in usage tracking and update on site switch (45176c9)

## Bug Fixes
- fix(adaptive-fallback-model): apply low reasoning effort to gpt-5 and willow-alpha models; add debug log (3d4ef38)
- fix(nautilus): remove workspace size startup check and adjust tool sets to prevent MCP startup failures (cef1370)
- fix(cli): correct ask mode reset via session run_state and fix AgentMode usage (c8b832b)
- fix(nemo/file-cache): skip directories when caching modified files in serve mode; add test (f28a7dc)
- fix(mcp): sanitize STDIO connection info to redact secrets in logs (1176d4d)
- fix(tool-permissions): rename getConfluencePages to getConfluencePage in v1 config (31df9c0)

## Engineering Improvements
- docs(rovodev-cli): update config docs adding Smart Tasks, session feedback, logging, and MCP settings (209aebb)
- docs(release-notes): add v0.12.14 release notes summarizing fixes and improvements (faf1e0e)
- chore(cli): bump version to 0.12.15 (901477a)
