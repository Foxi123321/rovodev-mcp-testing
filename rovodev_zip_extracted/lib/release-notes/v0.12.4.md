# Release Notes - v0.12.4

**Release Date:** October 10, 2025  
**Version:** 0.12.4  
**Previous Version:** 0.12.3

## New Features

- Config: auto-discover mcp.json by traversing parent directories up to repo root; handles inaccessible directories
- Code Nemo: collect artifacts on failure; enable reasoning by default and disable bash by default

## Bug Fixes

- Code Nautilus: make move_file more robust by validating directories and raising specific errors
- Run: initialize logging earlier to avoid UI clutter from pre-init logs
- Bash permissions: split commands on pipes for finer-grained permission checks
- Metrics: handle missing request count in responses API using message history fallback

## Engineering Improvements

- Logging: reduce noise and improve routing (including MCP STDIO); trim overly verbose site responses
- Tests: remove ad-hoc mock files in favor of shared mock_config fixture
- Docs: add v0.12.3 release notes
