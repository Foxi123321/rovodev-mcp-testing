# Release Notes: v0.13.9

**Release Date:** November 29, 2025  
**Previous Version:** v0.13.8

## Summary

This release focuses on improving user experience and system reliability with enhanced interrupt handling, UI improvements, and important dependency updates. Key highlights include better Ctrl+C behavior during tool permission prompts and the migration from requests to httpx library.

## New Features

- **Unified UI Formatting**: Improved spinner color consistency and standardized Ctrl+C copy formatting across the interface

## Bug Fixes

- **Enhanced Interrupt Handling**: Fixed Ctrl+C during tool permission prompts to prevent unexpected application exit
- **Async Shell Commands**: Resolved shell command execution behavior by making command handling fully asynchronous

## Engineering Improvements

- **HTTP Client Migration**: Replaced requests library with httpx across all packages for better async support and consistency
- **Async UI Architecture**: Converted shell command processing from synchronous to asynchronous execution for improved performance
