# Release Notes - v0.11.26

**Release Date:** September 30, 2025  
**Version:** 0.11.26  
**Previous Version:** 0.11.25

## New Features

- Nemo: add Anthropic native provider and migrate from classic; standardize provider naming
- LSP: switch to tag-based diagnostics, filter low-priority items, and map all to warnings
- CLI: refine site entitlement selection and Rovo Dev credits messaging; add detailed logging and error handling

## Bug Fixes

- Serve API: improve chat cancellation handling in v2 endpoints; update tests
- Bash permissions: prevent false positives in auto-deny regex (e.g., git branches) while still blocking dangerous commands
- Jira projects tool: prevent Atlassian API token leakage by resolving cloudId and using Cloud API
- CLI UI: ensure spinner doesn't cut off panels by printing a spacer line
- Site selection: use tenant_info to resolve cloudId; trust configured site and skip beta check

## Engineering Improvements

- CI/Pipelines: publish serve OpenAPI spec to Shipyard; add validation/provision steps for client code generation
- Docs: add v0.11.25 release notes
- Refactor: introduce ToolPermissionResolver; simplify bash/powershell and compressed tool handling; expand tests
