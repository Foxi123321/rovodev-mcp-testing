# Release Notes v0.13.1

**Release Date:** November 17, 2025
**Previous Version:** 0.13.0
**New Version:** 0.13.1

## New Features

- **Enhanced UI Components**: Improved input menu with multi-question support and custom input options for better user interaction
- **Model Context Limits**: Added support for @ to - conversion in model IDs for better compatibility
- **Serve Mode Improvements**: Added replay_end signal to indicate transition from replay to live streaming
- **LSP Analytics**: Restored agent analytics functionality by adding UserPromptTrackingCallback
- **PolyBench Utilities**: Added new utility scripts for diff filtering and result processing in evaluations

## Bug Fixes

- **Serve Mode Stability**: Fixed hanging issues on unsupported slash commands in serve mode
- **EOF Error Handling**: Improved error handling for invalid tool call arguments to prevent EOF errors
- **Todo Tool**: Relaxed validation requirements and improved user experience for todo functionality
- **ACP Interface**: Added proper formatting and kinds for all tool calls in ACP interface

## Engineering Improvements

- **Python Version Restriction**: Limited Python compatibility to 3.13 inclusive across all packages for better stability
- **Analytics Tracking**: Added CLI version to AI gateway headers for better tracking and debugging
- **Code Refactoring**: Renamed UserPromptTrackingCallback to AgentRunAnalyticsCallback for consistency
- **Cache Metrics**: Improved token count metrics with cache hit ratio and better edge case handling
- **UI Enhancements**: Added "Recommended" label to auto model option with updated command highlighting
