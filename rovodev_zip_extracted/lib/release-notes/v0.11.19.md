# Release Notes - v0.11.19

**Release Date:** September 12, 2025  
**Version:** 0.11.19  
**Previous Version:** 0.11.18

## New Features

- Add non-interactive mode for CLI (RDA-391)
- Include new command help content in AI-powered help (RDA-429)
- Sort commands alphabetically when typing (RDA-398)
- Add sandbox ID v2 support to analytics (RDA-423)
- Enable fetching custom fields in MCP Atlassian tools
- Add tool prefixes for commands and analytics output

## Security Fixes

- Remove dangerous tool from Nautilus

## Bug Fixes

- Correct system prompt ordering across agents
- Reduce Git AI stats collection frequency and duplication
- Correct field casing in Atlassian client for sandbox ID integration
- Prevent double slashes when constructing API Gateway URLs
- URI-encode PDF filenames when opening files

## Engineering Improvements

- Add Linux sandbox support and tests
- Include entitlement check payload in healthcheck details when failed (RDA-399)
- Improve feedback message formatting
- Run tests sequentially in pipelines
- Fix release tagging in pipelines
- Normalize "Jira" capitalization across UI and docs
